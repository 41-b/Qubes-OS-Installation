Qubes OS Installation 
📌 Overview
This project demonstrates the installation of Qubes OS using VMware virtualization on a Windows host system. It covers the complete setup process, system configuration, and troubleshooting.

🖥️ Environment
Host OS: Windows
Virtualization Tool: VMware
Guest OS: Qubes OS

📂 Project Structure
report/ → Full installation report
screenshots/ → Step-by-step installation proof
issues/ → Problems encountered and solutions

⚙️ Installation Summary
Download Qubes OS ISO
Configure VMware virtual machine
Allocate system resources
Install Qubes OS
Perform initial setup and configuration
